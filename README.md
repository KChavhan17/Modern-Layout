## 🖼️ Ronaldo 3 Image Layout


A simple HTML page that 
displays 3 images in a clean grid layout.


## 📁 File Structure


index.html — Single file,
58 lines, no dependencies needed.


## 🧱 How It Works


Top image fills full width
Bottom splits into 2 equal columns
Each image auto fits without distortion


## 🎨 CSS Used


🔲 Grid Layout
display: grid
grid-template-rows: 50vh 50vh
height: 100vh
➗ Bottom Split
grid-template-columns: 1fr 1fr


## 🖼️ Image Fit


object-fit: cover
object-position: center
width: 100%
height: 100%
🧹 Body Reset
margin: 0
font-family: Arial, sans-serif


## 🚀 How To Use


Open index.html in any browser
Replace the image src links with your own image URLs
No installation or setup needed


## 🌐 Browser Support
Works on Chrome, Firefox, Safari and Edge.
💡 Uses CSS Grid — a modern layout method. No old float or table tricks needed.
